Stack Data Structure

Stack is a linear data structure that follows a particular order in which the operations are performed. The order may be LIFO(Last In First Out) or FILO(First In Last Out). LIFO implies that the element that is inserted last, comes out first and FILO implies that the element that is inserted first, comes out last.

Push:

The push operation is used to add an element to the top of the stack.

stack.push(element);

Pop:

The pop operation is used to remove the element from the top of the stack.

element = stack.pop();

Peek(or Top):

The peek operation retrieves the element from the top of the stack without removing it.

element = stack.peek();
