Array and ArrayList Learning

Array:

An array is a fixed-size, sequential collection of elements of the same data type. Elements in an array are accessed using an index. Key characteristics include:
Indexing: Elements are accessed using an index (starting from 0).
Fixed Size: Size is defined at creation and cannot be changed.
Contiguous Memory: Elements are stored in contiguous memory locations.
Homogeneous Elements: Elements must be of the same data type.

ArrayList:

An ArrayList is a dynamic array implementation in Java, allowing for flexible sizing and additional methods. Key characteristics include:

Dynamic Sizing: Can dynamically resize itself.
Generic Type: Can store elements of a specified data type.
Random Access: Elements are accessed using an index.
Dynamic Memory Allocation: Internally adjusts size as needed.
Homogeneous Elements: Elements are typically of the same data type.

Difference:

The main difference lies in their flexibility.
Arrays have a fixed size, while ArrayLists can dynamically resize.
Arrays are generally more memory-efficient but less flexible, whereas ArrayLists offer more features but may consume more memory.

Adding and Deleting Elements:

Adding Element to Array: Check index bounds, then assign the new value to the specified index.

Adding Element to ArrayList: Use the add() method with the specified index to insert a new element.

Deleting Element from Array: Replace or remove the element at the desired index.

Deleting Element from ArrayList: Use the remove() method to remove an element by index or value.

Adding and Deleting Elements:

Adding Element to Array and ArrayList:

To add an element to an array, you check the index bounds and then assign the new value to the specified index. For example: myArray[2] = newValue;

To add an element to an ArrayList, use the add() method with the specified index to insert a new element. For example: myArrayList.add(2, newValue);

Deleting Element from Array and ArrayList:

To delete an element from an array, you replace or remove the element at the desired index. For example: myArray[2] = 0; // Replace with a default value

To delete an element from an ArrayList, use the remove() method to remove an element by index or value. For example: myArrayList.remove(2); // Remove element at index 2
